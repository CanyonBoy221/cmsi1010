# Problem Set 01: Functions and Game Programming

This problem set focuses on function creation and game logic implementation.

## Components

### Number Guessing Game
An interactive game where the program generates a random number and the player must guess it. Demonstrates:
- Random number generation
- User input handling
- Conditional logic and loops
- Game state management

### Function Exercises
Practice creating functions that perform specific operations based on input parameters. Includes function cardio exercises to build proficiency with:
- Function definitions and parameters
- Return values
- Function composition
- Problem-solving with functions
