# Problem Set 03: Card Games

Implementation of card game classes with comprehensive testing.

This problem set expands on object-oriented programming by creating a complete card game system. It includes:
- `Card` class with suits and ranks
- Card deck implementation and shuffling
- Poker game logic
- Comprehensive test suite using `cards_test.py`
- Classes cardio exercises for practice

The assignment reinforces concepts of dataclasses, class methods, testing, and game logic implementation.
