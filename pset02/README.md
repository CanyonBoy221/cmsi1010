# Problem Set 02: Data Structures and Mad Libs

This problem set focuses on working with Python data structures and string manipulation.

## Files

### data_structures_cardio.py
Practice exercises for working with Python's core data structures:
- Tuples
- Lists
- Sets
- Dictionaries

Includes unit tests to verify implementations. Run tests with:
```bash
python3 -m unittest data_structures_cardio.py
```

### mad_libs.py
An interactive mad libs program that generates random sentences by substituting words into templates. Demonstrates working with dictionaries, string replacement, and user input.
